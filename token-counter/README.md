# Token Counter

A simple utility created as an AI programming exercise to count tokens in text strings. Understanding token counts is essential for managing context windows and costs when working with LLM APIs.

## Purpose

- Learn about tokenization processes in AI models.
- Build tools to assist in AI application development.
- Exercise JavaScript/Node.js skills in a practical AI context.
