# Chunker

This project is a text processing utility built as part of an AI programming exercise. It focuses on the logic of splitting large strings into smaller segments (chunks) that can be efficiently processed by Large Language Models (LLMs).

## Purpose

- Practice handling text data for AI ingestion.
- Implement robust chunking strategies (e.g., character-based, semantic).
- Explore NestJS for building AI-related microservices.

## Getting Started

```bash
pnpm install
pnpm run start:dev
```
